Spring MVC에서 컨트롤러가 반환한 뷰 이름 문자열을 실제 뷰 객체로 변환하는 컴포넌트입니다.

## 핵심 특징

- [[DispatcherServlet]]이 컨트롤러의 반환값(뷰 이름)을 받아 ViewResolver에게 위임
- 설정된 prefix/suffix를 조합해 실제 뷰 파일 경로를 결정 (예: `/WEB-INF/views/` + `home` + `.jsp`)
- `InternalResourceViewResolver`(JSP), `ThymeleafViewResolver`(Thymeleaf) 등 뷰 기술에 따라 구현체가 다름
- `@ResponseBody` 또는 `ResponseEntity` 사용 시 뷰 리졸버를 거치지 않고 [[직렬화]] 후 HTTP 응답 본문에 직접 씁니다

## 핵심 정리

뷰 리졸버는 뷰 이름 → 뷰 객체 변환을 담당합니다. `@RestController` 계열은 뷰 리졸버를 사용하지 않습니다.
