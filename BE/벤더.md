**벤더(Vendor)** 는 **소프트웨어나 제품을 만들어서 판매/제공하는 회사**를 말합니다.

## DB 벤더 예시

|벤더(회사)|제품(DB)|
|---|---|
|Oracle|Oracle DB|
|Microsoft|SQL Server|
|Oracle (인수)|MySQL|
|PostgreSQL Global Development Group|PostgreSQL|

## 왜 이 용어를 쓰나?

JDBC 맥락에서 "DB 벤더"라고 하면, 각 데이터베이스를 만든 회사들이 **자기 DB에 맞는 드라이버를 직접 제공**한다는 의미입니다.

```
MySQL 쓸 때 → MySQL 벤더가 만든 드라이버 사용
Oracle 쓸 때 → Oracle 벤더가 만든 드라이버 사용
```
