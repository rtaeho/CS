웹사이트가 검색 엔진 결과에서 더 높은 순위에 노출되도록 최적화하는 작업입니다.

## 왜 필요한가?

- 검색 결과 상위에 있어야 방문자가 늘어남
- 광고비 없이 자연 유입(Organic Traffic) 확보
- 1페이지 밖의 결과는 거의 클릭되지 않음

## 검색 엔진 동작 방식

```
1. 크롤링: 봇이 웹사이트를 방문해 페이지 수집
2. 인덱싱: 수집한 내용을 분석해 저장
3. 랭킹: 검색어에 맞는 페이지를 순위대로 정렬
```

## SEO 주요 요소

|분류|항목|
|---|---|
|**콘텐츠**|키워드, 품질, 신선도|
|**기술적**|로딩 속도, 모바일 최적화, HTTPS|
|**HTML**|title, meta description, heading 태그|
|**링크**|백링크(다른 사이트에서 오는 링크), 내부 링크|

## 기술적 SEO 체크리스트

- 빠른 페이지 로딩 속도
- 모바일 반응형 디자인
- HTTPS 적용
- sitemap.xml 제출
- robots.txt 설정
- 시맨틱 HTML 사용
- 이미지 alt 태그

## 예시

```html
<!-- 좋은 예 -->
<title>2024 맥북 프로 리뷰 - 성능, 배터리 총정리</title>
<meta name="description" content="맥북 프로 M3 실사용 후기...">
<h1>2024 맥북 프로 상세 리뷰</h1>

<!-- 나쁜 예 -->
<title>글</title>
<div style="font-size:24px">맥북 리뷰</div>
```

## 개발자가 신경 쓸 부분

- SSR/SSG로 크롤러가 콘텐츠를 읽을 수 있게
- Core Web Vitals (LCP, FID, CLS) 최적화
- 구조화된 데이터(Schema.org) 적용
- 중복 콘텐츠 방지 (canonical 태그)
