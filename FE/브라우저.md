**브라우저(Browser)** 는 **웹페이지를 보여주는 프로그램**입니다.

## 대표적인 브라우저

|브라우저|만든 회사|
|---|---|
|Chrome|Google|
|Safari|Apple|
|Firefox|Mozilla|
|Edge|Microsoft|

## 브라우저가 하는 일

```
1. 주소 입력 (예: www.google.com)
2. 서버에서 HTML, CSS, JS 파일 받아옴
3. 화면에 보여줌 (렌더링)
```

## 좀 더 자세히

```
사용자가 URL 입력
       ↓
서버에 요청 (HTTP 통신)
       ↓
HTML, CSS, JS 파일 수신
       ↓
HTML → DOM 변환
CSS → CSSOM 변환
       ↓
Render Tree 생성
       ↓
Layout (위치/크기 계산)
       ↓
Paint (화면에 그리기)
       ↓
사용자가 웹페이지를 봄
```

## 일상 비유

|비유|설명|
|---|---|
|TV|방송국에서 보낸 신호를 영상으로 보여줌|
|브라우저|서버에서 보낸 코드를 웹페이지로 보여줌|

## 브라우저의 주요 기능

- 웹페이지 렌더링 (HTML, CSS 해석)
- JavaScript 실행 (엔진 내장)
- 네트워크 통신
- 데이터 저장 (쿠키, localStorage)
- 개발자 도구 제공

## 한줄 요약

브라우저 = **HTML, CSS, JS를 받아서 우리가 볼 수 있는 웹페이지로 바꿔주는 프로그램**
