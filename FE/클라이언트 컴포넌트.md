`'use client'` 지시어로 선언하며 브라우저에서 실행되는 React 컴포넌트로, [[서버 컴포넌트]]와 대비되는 개념이다.

## 핵심 특징

- 브라우저에서 실행: `useState`, `useEffect`, 이벤트 핸들러 등 모든 React 훅 사용 가능
- 파일 상단에 `'use client'` 선언 시 해당 파일과 하위 컴포넌트 모두 클라이언트 컴포넌트로 처리
- 클라이언트 번들에 포함되어 브라우저로 전송됨
- 서버 컴포넌트를 직접 import 불가 (서버 컴포넌트를 children props로 받는 방식은 가능)

## 핵심 정리

인터랙션(클릭, 입력, 실시간 업데이트)이 필요한 UI 부분에 적용하고, 데이터 페칭이나 정적 렌더링은 [[서버 컴포넌트]]에 위임하여 번들 크기를 최소화한다.
