**나중에 호출될 함수**입니다.
## 예시

```javascript
function greet(name) {
    console.log("안녕, " + name);
}

function process(callback) {
    callback("철수");  // 나중에 호출
}

process(greet);  // 출력: 안녕, 철수
```

`greet` 함수를 `process`에게 넘기고, `process`가 나중에 호출합니다.

---

## 비동기에서의 콜백

```javascript
// "1초 후에 이 함수 실행해줘"
setTimeout(() => {
    console.log("1초 지남");
}, 1000);
```

`() => { console.log("1초 지남") }` 이게 콜백입니다. setTimeout이 1초 후에 대신 호출해줍니다.

---

## 왜 쓰는가

```javascript
// 데이터 가져온 후에 처리하고 싶음
fetchData((data) => {
    console.log(data);  // 데이터 도착하면 실행
});
```

**"이 작업 끝나면 이거 실행해줘"** — 이런 상황에 콜백을 씁니다.

---

## 콜백의 어원

**Call**(호출) + **Back**(돌아와서)

"나중에 돌아와서 호출해줘"라는 의미입니다.
